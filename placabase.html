PLACA BASE

La placa base es una tarjeta de circuito impreso a la que se conectan los componentes que constituyen la computadora. A esta pieza se conecta la memoria RAM, la CPU, los discos duros y la fuente de alimentación, elementos esenciales para que un ordenador funcione a pleno rendimiento.

¿Qué es una placa base? Una placa base es un circuito impreso al que se conectan el resto de los componentes de un ordenador para que estos funcionen de manera óptima. También se las conoce con el nombre de placas madre. Su función es que todos los elementos esenciales de un PC cuenten con energía eléctrica para responder en todo momento de manera adecuada y coordinar los flujos de datos.

Zócalo:

Los zócalos son espacios o ranuras en la placa madre donde se insertan diferentes componentes como los zócalos para las memorias RAM, los zócalos de expansión para otras placas o el zócalo del microprocesador, etc. Es el dispositivo que se usa para conectar circuitos, sin realizar soldadura, lo cual evita someterlos a altas temperaturas, además de permitir el remplazo de un componente sin tener que someterlo a un proceso de desoldadura.

Chipset:

El chipset es un conjunto de chips conectados en la placa base, y se encarga de tareas como la gestión de periféricos externos, transferencia de datos entre la memoria y el microprocesador. El chipset es clave para determinar el rendimiento de un equipo, ya que si ordena el tráfico de datos de forma un poco lenta, todo el sistema va a ralentizarse.

Conectores de alimentación:

Los conectores de alimentación proporcionan energía a la placa base y a los componentes conectados a ella. Hay dos tipos principales de conectores de alimentación:

  1. Conectores de 24 pines: Estos conectores proporcionan energía a la placa base.
  2. Conectores de 8 pines: Estos conectores proporcionan energía adicional a la placa base, especialmente para los procesadores de alto rendimiento.

Ranuras o slots de memoria RAM:

Constituyen los conectores para la memoria principal del ordenador, es decir, la memoria RAM. Se encuentran en la placa madre de un ordenador y que permiten insertar módulos de memoria RAM. La función de estas ranuras es la de proporcionar acceso a la memoria RAM, que es un tipo de memoria volátil que se encarga de almacenar temporalmente los datos y programas que el procesador necesita para realizar sus tareas.

VRM:

Los VRM (Voltage Regulator Module) de la placa base consisten en una serie de componentes electrónicos que garantizan que el procesador o incluso la tarjeta gráfica reciban un suministro de energía limpio y, lo más importante, con el voltaje exacto y constante que necesitan.
Los encontramos tanto en ordenadores como en móviles, tablets y la mayoría de dispositivos electrónicos que requieran de una seguridad y conversión.

Ranuras o slots de expansión:

Elemento de conexión de la placa base de un ordenador que permite conectar a aquella una tarjeta adaptadora adicional, o de expansión, la cual suele realizar funciones de control de dispositivos periféricos adicionales, tales como monitores, impresoras o unidades de disco, entre muchos otros.

Conectores de entrada y salida:

Son todos aquellos que están directamente sobre la placa base y no son fácilmente accesibles desde el exterior. Esto hace que las placas base tengan una gran cantidad de puertos y conectores diferentes para distintas finalidades.

BIOS:

La ROM BIOS es de sólo lectura, por lo que no se puede escribir en ella. Contiene información grabada por el fabricante que no desaparece al desconectar el ordenador. Contiene las instrucciones para realizar el chequeo inicial del ordenador.

Tarjetas de sonido y de red:

Una tarjeta de sonido es una tarjeta de expansión para el ordenador que permite la entrada y salida de audio de forma regulada y controlada por un programa informático denominado driver.

Conectores SATA:

Un conector SATA es un tipo de interfaz que se utiliza para conectar discos duros, CD, DVD o Blu-ray a la placa base de un ordenador que tenga un puerto de conexión para SATA. Esto permite que este tipo de dispositivos de almacenamiento puedan realizar una transferencia de datos a una gran rapidez y de forma cómoda.

Conectores M.2:

M. 2 es un Factor de Forma utilizado para tarjetas de expansión y conectores que están montados internamente en un ordenador.

PLACA BASE

La placa base es una tarjeta de circuito impreso a la que se conectan los componentes que constituyen la computadora. A esta pieza se conecta la memoria RAM, la CPU, los discos duros y la fuente de alimentación, elementos esenciales para que un ordenador funcione a pleno rendimiento.

¿Qué es una placa base? Una placa base es un circuito impreso al que se conectan el resto de los componentes de un ordenador para que estos funcionen de manera óptima. También se las conoce con el nombre de placas madre. Su función es que todos los elementos esenciales de un PC cuenten con energía eléctrica para responder en todo momento de manera adecuada y coordinar los flujos de datos.

  1. Zócalo: Los zócalos son espacios o ranuras en la placa madre donde se insertan diferentes componentes como los zócalos para las memorias RAM, los zócalos de expansión para otras placas o el zócalo del microprocesador, etc.

  2. Chipset: El chipset es un conjunto de chips conectados en la placa base, y se encarga de tareas como la gestión de periféricos externos, transferencia de datos entre la memoria y el microprocesador.

  3. Conectores de alimentación

  4. Ranuras o slots de memoria RAM: Constituyen los conectores para la memoria principal del ordenador, es decir, la memoria RAM

  5. VRM

  6. Ranuras o slots de expansión

  7. Conectores de entrada y salida: Son todos aquellos que están directamente sobre la placa base y no son fácilmente accesibles desde el exterior.

  8. BIOS: La ROM BIOS es de sólo lectura, por lo que no se puede escribir en ella. Contiene información grabada por el fabricante que no desaparece al desconectar el ordenador. Contiene las instrucciones para realizar el chequeo inicial del ordenador.

  9. Tarjetas de sonido y de red: Una tarjeta de sonido es una tarjeta de expansión para el ordenador que permite la entrada y salida de audio de forma regulada y controlada por un programa informático denominado driver.

  10. Conectores SATA: Un conector SATA es un tipo de interfaz que se utiliza para conectar discos duros, CD, DVD o Blu-ray a la placa base de un ordenador que tenga un puerto de conexión para SATA. Esto permite que este tipo de dispositivos de almacenamiento puedan realizar una transferencia de datos a una gran rapidez y de forma cómoda.

  11. Conectores M.2: M. 2 es un Factor de Forma utilizado para tarjetas de expansión y conectores que están montados internamente en un ordenador.
